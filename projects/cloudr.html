Development- Cloudr

Ruby on Rails and D3.js

cloudr.herokuapp.com

Github Repository

Rails/D3 application to present popular Reddit articles in a more aesthetically pleasing way. Presents popular Reddit articles as a word cloud using D3.js. Links each article by customizing previous D3.js word cloud with various Javascript functions. Scrapes popular web articles from Reddit using Nokogiri.
